## Домашнее задание по теме Микросервисы - Масштабирование ##  

### Задание 1 - Кластеризация ###  
Задачу по предъявляемым требованиям в принципе можно решить как с помощью Docker swarm, так и и с помощью Kubernetes.  
В решении Kubernetes или платформ управления, базирующихся на Kubernetes, таких как:  
 - Astra Боцман
 - Orion Nova
 - Flant Deckhouse  
уже, обычно содержатся инструменты для:
- обнаружения и маршрутизации сервисов,
- автоскэйлинга, 
- хранения параметров серсивос и секретов,
- разграничения доступов  к сервисам  
В случае необходимости для решения отдельных задач можно добавить продукты:  
- Apache ApiSix - для маршрутизации сервисов
- Hashicorp Consul - для хранения параметров сервисов и реализации Service Registry
- Hashicorp Vault/Openbao/Orion StarVault - для безопасного хранения секретов
Таким образом, можно выбрать, например, решение от российской компании Orion, которое активно развивается, обеспечено локальной технической поддержкой и предоставляет как стандартные функции Kubernetes, так и дополнительные инструменты, в частности StarVault для хранения секретов.
